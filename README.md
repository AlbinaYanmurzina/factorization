## О проекте

ВКР по теме: реализация и сравнительное исследование алгоритмов факторизации целых чисел на основе квадратичного решета.

**Цель:** программная реализация и сравнительный анализ эффективности экспоненциальных алгоритмов (методы Полларда) и субэкспоненциального метода квадратичного решета.

## Реализованные алгоритмы

| Ключ API | Алгоритм | Сложность |
|---|---|---|
| `pollard_rho` | ρ-метод Полларда | Экспоненциальный |
| `pollard_p1` | (p−1)-метод Полларда | Экспоненциальный |
| `qs_basic` | Базовое квадратичное решето (Диксон) | Субэкспоненциальный |
| `qs_optimized` | Оптимизированное QS (просеивание) | Субэкспоненциальный |
| `qs_lpv` | QS с вариацией больших простых (LPV) | Субэкспоненциальный |
| `qs_mpqs` | Множественные полиномы (MPQS) | Субэкспоненциальный |
| `qs_mpqs_parallel` | Параллельный MPQS | Субэкспоненциальный |
| `qs_auto` | Автовыбор оптимального QS | — |

## Установка и запуск

### 1. Создание и активация виртуального окружения

```bash
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/macOS
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск backend (FastAPI)

```bash
cd backend
python main.py
# API доступен на http://127.0.0.1:8000
# Документация: http://127.0.0.1:8000/docs
```

### 4. Запуск frontend (Streamlit)

```bash
cd frontend
streamlit run Главная.py
```

### 5. Запуск тестов

```bash
python -m pytest tests/ -v
```
